# Jarvis - AI Desktop Assistant for Windows

Your personal AI-powered voice assistant for Windows.

## Features
- Voice Interaction - Speak naturally and Jarvis responds
- Text Mode - Chat with Jarvis in text mode
- Multilingual - Any language you want
- Open Files and Apps - Open any application or file
- Web Research - Searches and summarizes results
- Window Management - Open and close windows
- Weather - Real-time weather updates
- Time and Date - Always know the time
- System Info - CPU, RAM, battery, disk usage
- YouTube Stats - Fetch statistics
- WhatsApp - Send messages via voice or text
- Media Control - Control your media
- Calendar and Reminders - Set reminders
- Screen Vision - Analyze your screen

## Installation
1. Clone the repo
2. Run setup.ps1
3. Run python main.py
